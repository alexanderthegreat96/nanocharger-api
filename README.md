## Nano-Charger API BETA

RESTful API implementation that implements multiple methods for query-ing charging stations based on location. 


### Getting started

 - Pull the repository
 - Edit .env  variables to suit your needs
 - docker-compose up --build
 - Navigate to [docs] to check the documentation and import the POSTMAN collection

I highly recomend importing .sql file [nanocharger.sql] as it has enough data that has been tested in order to test the query-ing functionality
